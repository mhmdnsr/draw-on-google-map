# Release Notes - v2.0.0

`v2.0.0` is a full modernization release for `draw-on-google-map`.

Highlights:
- TypeScript-based refactor with safer lifecycle handling.
- New tools: polyline, circle, rectangle.
- Events API (`on`/`off`) and serialization (`exportData` / `importData`) with JSON + GeoJSON.
- Stronger multi-instance isolation and explicit `destroy()` cleanup method.
- Updated package metadata and framework examples using canonical package naming.

For the complete change list, see [CHANGELOG.md](./CHANGELOG.md).
